# 艾伦爬虫程序 使用说明

### 本程序仅供技术交流 

+ 先安装依赖包

  ```python
  pip install -r requirements.txt
  ```

+ 源地址没改请在后面加 -i https://pypi.douban.com/simple

+ star_page.py是程序主入口 (运行前请检查依赖包有无问题)

### 程序的设计

+ 本程序界面采用tkinter制作
+ 基于selenium进行数据爬取
+ 基于numpy和matplotlib进行数据分析及可视化
